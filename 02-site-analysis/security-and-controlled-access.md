# Security and Controlled Access

## Objective

Identify areas of the site that require controlled access and security planning.

## High-Security Functions

- Diamond cutting house
- Gold refinery
- Jewellers hub
- Secure storage
- Secure deliveries and collections

## Security Planning Considerations

- Public access must not flow through high-security areas.
- Secure deliveries may require a separate entrance.
- Staff access may need checkpoints.
- Visitor access to high-security areas must be controlled.
- Secure areas should have clear boundaries.

## Student Tasks

- Mark high-security areas on the zoning concept.
- Identify where access control points may be needed.
- Identify possible security risks in the initial layout.
- Prepare questions for Cyber Security and Ben.
