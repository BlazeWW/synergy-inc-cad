# First Concept Board Requirements

Each Architectural Draughting student or group must prepare a first concept board after completing site analysis.

## Required Content

The board should include:

- Project name: Synergy Inc.
- Site reference and 24-hectare note
- Location reference from Google Maps
- Main assumptions
- Major proposed precincts
- Public/private/restricted zoning
- Vehicle and pedestrian movement concept
- Parking and service access ideas
- Security-sensitive areas
- Questions requiring confirmation

## Format

Preferred formats for review:

- PDF
- PNG
- JPG

Save exports in:

`04-exports-for-review/`
