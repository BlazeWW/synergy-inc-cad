## Expected Output After Working Through These Resources

Students should be able to:

- Use GitHub Issues to track draughting tasks.
- Commit Markdown notes and exported review files.
- Understand basic AutoCAD 2D drafting workflow.
- Begin site analysis from the Google Maps reference.
- Prepare public, private, service, noisy, quiet, and restricted-zone diagrams.
- Export concepts to PDF or image format for review.
